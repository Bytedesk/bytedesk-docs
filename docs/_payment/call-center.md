# Call Center Pricing

The call center module is delivered as an add-on to the intelligent customer service suite. Pricing combines the software license, the number of concurrent voice channels, and optional telephony services.

## Pricing Components

| Component | Description | Billing Model |
| --- | --- | --- |
| Base License | Unlocks IVR, queue management, quality monitoring, and agent desktop capabilities in the [Call Center module](../plugins/freeswitch.md). | One-time perpetual license or annual subscription |
| Seat Pack | Named agents that can simultaneously log into the voice console. Sold in bundles of 5 seats. | Annual subscription |
| Concurrent Channel Pack | SIP concurrent calls/IVR ports. Determines how many calls can be handled at the same time. | One-time activation + annual maintenance |
| Telephony Access | Optional PSTN trunks, SMS notifications, or cloud recording storage. | Pay-as-you-go through partnered carriers |
| Professional Services | Deployment, voice flow design, data migration, and integration with CRM/CTI. | Time & material (¥2,000 / consultant / day) |

## Deployment and Support

1. **Cloud deployment**: Provisioned in 1-2 business days. Includes managed FreeSWITCH cluster, automatic scaling, and 99.9% SLA.
2. **Private deployment**: Delivered as Docker/Kubernetes package with automation scripts. Onboarding typically takes 3-5 working days.
3. **Source code delivery**: Provided via private Git repository plus a half-day knowledge transfer session.

Each option includes:

- Version upgrades for one year (renewable at 15%/year afterward)
- Access to the technical support portal and priority ticket handling
- Remote assistance sessions for troubleshooting (up to 10 hours/year)

For a tailored quotation, please prepare the expected number of agents, peak concurrent calls, required regions, and carrier preferences before contacting our sales team.
