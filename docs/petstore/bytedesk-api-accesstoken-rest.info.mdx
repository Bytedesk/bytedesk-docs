Bytedesk（微语）系统的真实接口文档，基于 OpenAPI 3.0。本文档以 AccessToken 登录（免二次输入账号密码）为核心，
并说明使用 Bearer Token 调用 REST API 的方式。

- 官网：https://www.weiyuai.cn
- 源仓库：https://github.com/bytedesk/bytedesk
- 后端 OpenAPI 源定义：https://api.weiyuai.cn/v3/api-docs

术语说明：
- AccessToken：在管理后台生成的第三方登录凭证；用于 /agent 与 /admin 的免登。
- Bearer Token：登录后用于调用 /api/* 接口的鉴权令牌（Authorization: Bearer &lt;token&gt;）。
