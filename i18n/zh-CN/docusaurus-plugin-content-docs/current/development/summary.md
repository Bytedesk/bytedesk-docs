# 会话小结

## 功能简介

在服务结束或阶段节点，对本次会话进行结构化沉淀，帮助“快速回顾、顺畅交接、精准检索”，提升团队协作与复盘效率。

## 你能做什么

- 使用模板快速编写小结，保证信息结构统一
- 支持自定义字段（如问题类型、影响范围、根因、下一步）
- 自动保存草稿，历史可追溯，可全文搜索
- 与会话、客户、工单自动关联，形成统一上下文

## 快速上手

1. 启用与配置：在管理后台开启“会话小结”，按需配置模板与字段。
2. 编写小结：在会话右侧面板或通过快捷键打开小结，填写关键信息。
3. 建议结构：问题背景 → 处理过程 → 结论与下一步。
4. 保存与关联：保存后自动与当前会话、客户、相关工单进行关联，便于检索与统计。

## 最佳实践

- 统一口径：采用固定的三段式结构，便于快速浏览与培训新人。
- 关键词：在小结中加入关键标签（如“退款”“Bug”“权限”）方便检索。
- 链接资料：附上相关工单、知识库文章或外部文档链接，提升复用度。
