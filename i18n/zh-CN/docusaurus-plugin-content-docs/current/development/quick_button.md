# 快捷按钮

快捷按钮用于把「常用操作」固定成一个按钮，客服在对话工具栏中点击即可快速发送对应的结构化消息或执行对应动作。

适用场景：

- 反复发送同类内容（如表单/引导/链接/商品/工单相关操作）
- 需要“点一下就发/就触发”，减少重复输入与出错

使用流程（推荐顺序）：创建 → 选择类型并配置 → 绑定到工具栏 → 测试 → 在客服端使用

## 添加快捷按钮

在管理端创建一个新的快捷按钮条目，并填写名称等基础信息。

![quick_button_add](/img/quick_button/quick_button_add.png)

## 快捷按钮类型

快捷按钮“类型”决定了点击后发送/触发的内容结构。选择类型后，页面会显示对应的配置项（不同类型配置项不同）。

![quick_button_type](/img/quick_button/quick_button_type.png)

## 绑定快捷按钮到工具栏

快捷按钮创建完成后，需要绑定到对话工具栏，客服端才会展示并可点击使用。

![quick_button_toolbar](/img/quick_button/quick_button_toolbar.png)

## 测试快捷按钮

建议在测试对话中点击一次，确认发送内容/触发行为符合预期。

![quick_button_test](/img/quick_button/quick_button_test.png)

## 客服端查看快捷按钮消息

客服端点击工具栏中的快捷按钮后，会在对话中看到对应的消息内容（不同类型展示不同）。

![quick_button_agent](/img/quick_button/quick_button_agent.png)

## 常见问题

- 客服端看不到按钮：通常是未绑定到工具栏，或当前账号/组织无权限。
- 点击无效果：检查按钮类型配置是否完整，以及是否在正确的会话/渠道中测试。
