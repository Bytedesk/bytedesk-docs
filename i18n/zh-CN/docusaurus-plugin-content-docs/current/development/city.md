# 城市管理及手机号归属地查询

本页面面向非技术人员，介绍“城市管理”“手机号归属地查询”和“IP 归属地查询”三个功能，帮助你快速上手。

## 这个模块能做什么
- 维护城市数据：新增、编辑、删除、启用/禁用城市信息。
- 一键初始化：从内置数据导入城市列表。
- 一键重置：清空全部城市数据（不可恢复）。
- 手机号归属地查询：输入手机号或号段，查看归属地与运营商。
- IP 归属地查询：输入 IP 地址，查看归属地信息。

## 快速上手
### 1. 城市管理
1. 打开城市管理页面。
2. 点击“新增”创建城市；或点击“编辑”修改已有城市。
3. 需要删除时，点击“删除”并确认。
4. 若需快速导入城市数据，点击“一键初始化”。
5. 如需清空城市数据，点击“一键重置”（不可恢复，请谨慎）。

![city](/img/city/city.png)

### 2. 手机号归属地查询
1. 点击“手机号归属地”。
2. 输入 11 位手机号，或输入前 7 位号段。
3. 点击“查询”，查看结果。

![city](/img/city/city_phone.png)

### 3. IP 归属地查询
1. 点击“IP 归属地”。
2. 输入 IPv4 或 IPv6 地址。
3. 点击“查询”，查看结果。

![city](/img/city/city_ip.png)

## 查询结果说明
- 号码：你输入的手机号或号段。
- 省份/城市：号码所属地区。
- 邮编/区号：该地区常用邮编和区号。
- 运营商：号码所属运营商（如移动、联通、电信）。
- IP 地址：你输入的 IP。
- 原始位置：IP 归属地的原始字符串。
- 国家/区域/省份/城市/运营商：从原始位置中解析出的字段。

## 规则与限制
- 仅支持 7-11 位数字输入。
- 无数据时会提示“未返回数据”或“未找到”。
- 导出功能在生产环境可能不可用（如有需要请联系管理员）。
- IP 查询仅支持合法的 IPv4/IPv6 地址。

## 常见问题
**Q: 查不到结果怎么办？**
可能是号码无归属数据或输入格式不正确，请确认是 7-11 位数字。

**Q: 会不会修改系统数据？**
手机号归属地查询只做查询，不会改动任何数据。

**Q: IP 归属地查询会修改系统数据吗？**
不会，只做查询。

**Q: 必须输入完整手机号吗？**
不必须。输入 11 位手机号或前 7 位号段都可以。

**Q: IP 查询支持 IPv6 吗？**
支持，请输入合法的 IPv6 地址。
