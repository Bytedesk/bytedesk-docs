# Intercom 对比

Intercom 是一个客户关系管理平台，允许企业与客户进行实时沟通。它提供了一个统一的平台，用于团队之间的沟通和协作。Intercom 通过提供客户支持、客户关系管理、客户参与和客户分析来帮助企业。
