# Quick Button

快捷按鈕用於把「常用操作」固定成一個按鈕，客服在對話工具列中點擊即可快速送出對應的結構化訊息或觸發對應動作。

適用情境：

- 重複送出同類內容（如表單/引導/連結/商品/工單相關操作）
- 需要「點一下就送出/就觸發」，減少重複輸入與出錯

使用流程（建議順序）：建立 → 選擇類型並設定 → 綁定到工具列 → 測試 → 在客服端使用

## 新增快捷按鈕

在管理端新增一個快捷按鈕條目，並填寫名稱等基礎資訊。

![quick_button_add](/img/quick_button/quick_button_add.png)

## 快捷按鈕類型

快捷按鈕「類型」決定點擊後送出/觸發的內容結構。選擇類型後，頁面會顯示對應的設定項（不同類型設定項不同）。

![quick_button_type](/img/quick_button/quick_button_type.png)

## 綁定快捷按鈕到工具列

快捷按鈕建立完成後，需要綁定到對話工具列，客服端才會顯示並可點擊使用。

![quick_button_toolbar](/img/quick_button/quick_button_toolbar.png)

## 測試快捷按鈕

建議在測試對話中點擊一次，確認送出內容/觸發行為符合預期。

![quick_button_test](/img/quick_button/quick_button_test.png)

## 客服端查看快捷按鈕訊息

客服端點擊工具列中的快捷按鈕後，會在對話中看到對應的訊息內容（不同類型展示不同）。

![quick_button_agent](/img/quick_button/quick_button_agent.png)

## 常見問題

- 客服端看不到按鈕：通常是未綁定到工具列，或目前帳號/組織無權限。
- 點擊無反應：檢查按鈕類型設定是否完整，並確認在正確的會話/渠道中測試。
