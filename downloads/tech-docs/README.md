## API 接口概览

本系统提供 **2418** 个 API 接口，涵盖以下功能模块：

- **用户认证**: 登录、注册、令牌管理
- **用户管理**: 用户 CRUD、角色分配、权限管理
- **组织管理**: 组织架构、部门管理
- **即时通讯**: 单聊、群聊、消息管理
- **客服系统**: 访客管理、工作组、路由规则
- **AI 智能体**: 机器人、对话管理、Prompt 管理
- **知识库**: 文章、分类、FAQ
- **工单系统**: 工单创建、流转、SLA
- **工作流**: 流程定义、节点管理、触发器
- **文件管理**: 文件上传、下载、预览
- **系统配置**: 设置、参数、配置管理

详细接口文档请查看 [06-API接口文档](./06-API接口文档.md)

## 技术栈

- **后端框架**: Spring Boot 3.5.8
- **Java 版本**: JDK 17+
- **数据库**: MySQL / PostgreSQL / H2 / Oracle / KingBase
- **缓存**: Redis + Caffeine
- **ORM**: Spring Data JPA
- **API 文档**: SpringDoc OpenAPI 3.0
- **AI 集成**: Spring AI (支持多家 AI 提供商)
